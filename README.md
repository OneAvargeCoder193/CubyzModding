# Cubyz Wasm Example Mod
## How to add to game:
Install this [fork of cubyz](https://github.com/OneAvargeCoder193/Cubyz/tree/wasm)
In this project, run a zig compiler version 0.15.1 or higher, and place the .wasm file in zig-out\bin into the mods folder in the cubyz fork
Run cubyz, and the mod should be installed.
